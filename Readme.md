ขาจ่ายสัญญาณและเซนเซอร์ (I/O Pins)
PA0 ➡️ ต่อเข้า AIN1 (บอร์ดแดง DRV8833)

PA1 ➡️ ต่อเข้า AIN2 (บอร์ดแดง DRV8833)

PA6 ➡️ ต่อเข้าสาย C1 สีเขียว (Encoder ท้ายมอเตอร์)

PA4 ➡️ ต่อเข้าสาย C2 สีเหลือง (Encoder ท้ายมอเตอร์)

PC0 ➡️ ต่อเข้า ขากลาง (R-เกือกม้า)

(PC13 เป็นปุ่มสีฟ้าบนบอร์ด กดใช้งานได้เลย ไม่ต้องต่อสายเพิ่มครับ)

⚡ ขาจ่ายไฟเลี้ยง (Power Pins)
5V ➡️ ต่อเข้า VM (บอร์ดแดง DRV8833)

3.3V ➡️ ต้องพ่วงแยกไป 3 จุด:

STBY (บอร์ดแดง DRV8833)

VCC สายสีดำ (Encoder ท้ายมอเตอร์)

ขาซ้ายสุด (R-เกือกม้า)

GND ➡️ ต้องพ่วงแยกไป 3 จุด:

GND (บอร์ดแดง DRV8833)

GND สายสีน้ำเงิน (Encoder ท้ายมอเตอร์)

ขาขวาสุด (R-เกือกม้า)
